# Contact Management System

## About the Project
This project is a Python-based Contact Management System developed as part of Week 3 of my internship.

The system allows users to store, manage, and organize contact details efficiently using Python. It demonstrates the use of dictionaries, functions, and file handling for real-world applications.

---

## Features
- Add new contacts with details (name, phone, etc.)
- Search contacts using name (partial matching)
- Update existing contact information
- Delete contacts
- Display all saved contacts
- Data stored using file handling (SQLite/JSON)

---

## Technologies Used
- Python
- File Handling (JSON / SQLite)
- VS Code

---

## How to Run

### Requirements:
- Python 3 installed

### Steps:
1. Download or clone this project  
2. Open terminal in project folder
